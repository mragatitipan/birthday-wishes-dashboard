# 🎂 Birthday Wishes Dashboard

Interactive birthday greeting website built with HTML, CSS, and JavaScript featuring a modern responsive UI and elegant user experience.

## ✨ Features

* 💌 Birthday wishes submission form
* 📊 Admin dashboard to view all submitted greetings
* 🎵 Automatic background music playback
* 📷 Multiple photo upload support
* 📱 Fully responsive design for mobile & desktop
* 🌸 Elegant cinematic birthday-themed interface
* ⚡ Smooth animations and interactive effects
* 📨 Real-time greeting counter display

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 4
* Animate.css

## 📷 Main Functionalities

### User Features

* Submit birthday wishes
* Upload photos with greetings
* Interactive and animated UI experience
* Responsive layout across devices

### Admin Dashboard

* View all greetings submitted by users
* Monitor birthday messages and uploaded photos
* Manage guest interactions easily

## 🚀 Installation

Clone repository:

```bash
git clone https://github.com/mragatitipan/birthday-wishes-dashboard.git
```

Open project folder and run using local server such as XAMPP or Live Server.
